The Science Behind Hridaya

An intricate look at the layered systems powering our non-invasive, medical-grade monitoring wristband.

System Breakdown

build

Structure Mechanical System

Outer Fabric Strap

  • Provides secure and comfortable fit
  • Breathable textile suitable for infant skin
  • Integrated Velcro closure for adjustability

Inner Silicone Padding

  • Soft cushioning layer between device and skin
  • Maintains consistent sensor contact
  • Prevents irritation and pressure points

Protective Housing (ABS/PC)

  • Encases all electronic components securely
  • Shock-resistant, lightweight with rounded edges
  • Separates inner skin surface from exterior
memory

Electronics Core System

Compact PCB

The Central Control Unit of the device.

  • Printed Circuit Board integrating all components
  • Acts as the system's processing and communication hub

Sub-Components

  • ESP32 Microcontroller: Handles heavy processing and secure BLE (Bluetooth Low Energy) communication.
  • Thin high-capacity battery unit attached via standard connector.
sensors

Sensors Skin-Contact System

PPG Sensor

Optical Heart Rate Sensor.

  • Uses light-based detection to measure pulse
  • Positioned directly on skin contact surface
  • Requires stable and gentle pressure

Skin Temperature Sensor

Embedded Contact Sensor.

  • Accurately measures surface temperature of the skin
  • Positioned directly beside the PPG sensor array for combined reading
vibration

Output Feedback System

LED Indicator

Visual Status Output.

  • Displays device status (power, active connection)
  • Positioned on the outer strap surface for clear visibility

Vibration Motor

Haptic Feedback Unit.

  • Provides gentle tactile alerts to parents or subtle soothing patterns
  • Positioned specifically away from sensors to prevent mechanical interference
